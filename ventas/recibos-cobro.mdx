# Recibos de Cobro

## Introducción

Los recibos de cobro registran los pagos recibidos de tus clientes y mantienen el balance de cuentas por cobrar actualizado.

## Crear un recibo

1. Ve a **Ventas > Recibos de Cobro**
2. Haz clic en **Nuevo Recibo**
3. Selecciona el cliente
4. Elige las facturas pendientes
5. Indica el monto y la forma de pago
6. Guarda

## Aplicación automática

El sistema puede **auto-aplicar** pagos a las facturas pendientes para ahorrar tiempo.

## Retenciones (si aplica)

Si el cliente requiere retenciones, puedes aplicarlas desde el recibo para que el saldo quede correcto.

## Impresión y exportación

Desde el detalle puedes:

- **Imprimir** el recibo
- **Exportar** a Excel o PDF

## Recomendaciones

- Revisa que el monto aplicado coincida con las facturas seleccionadas
- Usa la auto-aplicación cuando el pago cubre varias facturas

## Por qué es fácil

- Pagos aplicados en segundos
- Control claro de facturas pendientes
- Recibos listos para imprimir
