# Punto de venta

## Qué es este módulo

El **Punto de venta** (POS) está optimizado para cobrar en persona de forma ágil: escanear o buscar productos, aplicar descuentos si corresponde, cobrar con distintos medios de pago y emitir el comprobante al cliente.

## Uso típico

1. Abre el módulo desde el icono **Punto de Venta** en la barra lateral.
2. Selecciona la **caja** o turno con el que vas a operar, si el sistema lo solicita.
3. Agrega líneas al ticket (productos o servicios).
4. Indica el **método de pago** (efectivo, tarjeta, transferencia, etc.).
5. Finaliza la venta para generar el documento y, si aplica, abrir cajón o impresión.

## Reportes y cierres

Según la configuración de tu organización, podrás consultar **reportes de ventas del POS** y realizar **cierres de caja** para cuadrar lo cobrado con lo declarado.

## Recomendaciones

- Cierra caja con la periodicidad que defina tu negocio (turno, día) para detectar diferencias a tiempo.
- Si algo no cuadra, revisa primero los tickets anulados o pendientes antes del cierre.
