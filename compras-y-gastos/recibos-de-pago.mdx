# Recibos de pago

## Crear un recibo de pago

1. Ve a **Compras y Gastos → Recibos de pago → Nuevo**
2. Selecciona el suplidor (o el tipo de pago especial, por ejemplo [préstamo a empleados](/recursos-humanos/prestamos))
3. Elige los documentos pendientes (compras, gastos) y los montos a aplicar
4. Indica formas de pago (efectivo, transferencia, cheque, pasarela, etc.)
5. Aplica **retenciones** si corresponden (el sistema filtra opciones según RST y configuración)
6. Guarda

## Retenciones

Solo verás tipos de retención compatibles con el régimen del suplidor y el de tu organización. Revisa el [reporte de retenciones](/fiscal/reporte-retenciones) después de cerrar el mes.

## Impresión y exportación

Desde el detalle puedes imprimir el recibo y exportar cuando lo necesites para el suplidor o el archivo contable.

## Recomendaciones

- Verifica que la suma de métodos de pago coincida con el total aplicado
- Usa una sola transferencia real del banco y un solo recibo para no duplicar movimientos
