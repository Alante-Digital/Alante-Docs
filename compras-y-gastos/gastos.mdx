# Gastos

## Crear un gasto

1. Ve a **Compras y Gastos → Gastos → Nuevo gasto**
2. Indica descripción, monto, fecha, moneda y centro de costos si aplica
3. Asocia un **suplidor** cuando exista documento de proveedor (opcional en algunos flujos)
4. Revisa retenciones y totales
5. Guarda y, si tu flujo lo requiere, **aprueba**

## Con o sin suplidor

- **Con suplidor**: entra a cuentas por pagar y se paga con [recibo de pago](/compras-y-gastos/recibos-de-pago)
- **Sin suplidor**: útil para egresos internos; según configuración puede salir directo de caja/banco

## Gasto menor

Puedes registrar operaciones como **gasto menor** cuando corresponda a la normativa fiscal. Esas operaciones también se gestionan o revisan desde [Fiscal → Gastos menores](/fiscal/gastos-menores).

## Proveedores del extranjero

En compras/gastos a suplidores extranjeros el sistema puede mostrar vista previa de retenciones aplicables (por ejemplo ISR). Confirma el régimen del contacto antes de pagar.

## Caja chica

Los gastos rápidos desde una [caja chica](/cajas-y-bancos/cajas) generan el gasto y la salida de efectivo juntos.

## Recomendaciones

- Aprueba antes de pagar para separar roles (quien registra vs quien autoriza)
- Adjunta o referencia el comprobante en notas para auditoría
