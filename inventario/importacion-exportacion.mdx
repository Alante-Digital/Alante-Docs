# Importación y exportación de productos

## Exportar productos y servicios

Puedes exportar el catálogo a un archivo **CSV** (compatible con Excel) desde:

**Inventario → Productos y servicios → Exportar**

**Opciones habituales:**

- Tipo: solo productos, solo servicios o ambos
- Categoría (opcional)
- Incluir existencias por almacén

**Qué trae el archivo exportado**

- Un **identificador único de cada producto** generado por Alante (sirve para que el sistema reconozca el mismo artículo si vuelves a importar y quieres actualizarlo en lugar de crear duplicados)
- Código interno (SKU), nombre, tipo y estado
- Categoría, código de barras y unidad de medida
- Precio de venta, costo, margen e ITBIS
- Indicadores de si es vendible, comprable, si maneja existencias y si permite existencia negativa
- Cantidades por almacén (si elegiste incluir existencias)

## Importar productos y servicios

Desde **Inventario → Productos y servicios → Importar** puedes cargar un archivo CSV.

### Plantilla

Descarga la **plantilla oficial** desde la misma pantalla de importación. Así las columnas coinciden con lo que el sistema espera y reduces errores.

### Cómo actualizar productos existentes

Si el archivo incluye la columna con el **identificador del producto** (el mismo valor que viste en una exportación previa de Alante), el sistema **actualiza** ese registro en lugar de crear uno nuevo.

### Cómo evitar duplicados

Si **no** incluyes el identificador y el **código de barras** ya existe en el catálogo, esa fila se **omite** para no duplicar artículos.

### Existencias

- La cantidad de existencia indicada en el archivo suele aplicarse al **almacén principal**.
- Los **servicios** no deben llevar cantidad en existencia.

### Validación antes de importar

Antes de confirmar, el sistema revisa el archivo y te avisa si hay problemas de formato, datos inválidos o filas duplicadas entre sí.

## Por qué es fácil

- Plantilla lista para descargar
- Mensajes claros si algo falla en la importación
- Exportación útil para respaldos o edición masiva en Excel
